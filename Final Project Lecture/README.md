# HobbyCycle - Marketplace Barang Hobi Bekas

Proyek ini adalah tugas Final Project Web Programming untuk membuat platform jual beli barang hobi (TCG, Game, dll).

## Fitur Minggu 1
- Registrasi User (Password Hashing)
- Login System
- Dashboard Sederhana

## Cara Menjalankan (Installation)
1. Download repository ini atau clone.
2. Pindahkan folder `hobbycycle` ke dalam folder `htdocs` di XAMPP.
3. Nyalakan Apache dan MySQL di XAMPP Control Panel.
4. Buka `localhost/phpmyadmin`, buat database baru dengan nama `hobbycycle_db`.
5. Import file `database.sql` yang ada di dalam folder ini ke database tersebut.
6. Buka browser dan akses `localhost/hobbycycle/index.php`.

## Tech Stack
- HTML, CSS
- PHP Native
- MySQL
