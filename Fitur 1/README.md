Week 1: Fondasi & Autentikasi
Fokus pada keamanan akses pengguna dan struktur dasar database.
Register: Pendaftaran akun baru dengan enkripsi password (`password_hash`).
Login: Sistem login menggunakan `Session` untuk menyimpan status pengguna.
Logout: Menghapus sesi aman.
Security: Penerapan `mysqli_real_escape_string` untuk mencegah SQL Injection.
