Fokus pada logika pembelian dan manajemen stok.
 Buy Logic: Sistem pembelian yang otomatis mengubah status barang dari `Available` menjadi `Sold`.
 Stock Control: Barang yang sudah dibeli otomatis hilang dari halaman pencarian.
 Transaction Log: Mencatat ID pembeli, ID barang, dan tanggal transaksi di database.
 Order History: Halaman "Barang Belanjaan Saya" untuk melihat riwayat barang yang sudah dibeli.
